Planilha de Precificação para Confeitaria (Doce-u)
📌 Sobre o Projeto
Esta é uma ferramenta criada no Excel para calcular o custo exato de receitas e definir o preço de venda de forma automática, garantindo a margem de lucro correta e padronizando a precificação.

🛠️ Conhecimentos Aplicados
Neste projeto, utilizei o Excel para resolver um problema real de negócios, aplicando as seguintes funções e conceitos:

PROCV: Para buscar automaticamente o custo unitário de cada ingrediente no banco de dados.

SOMASES: Para calcular o custo total de uma receita, consolidando todos os itens da ficha técnica.

Matemática e Porcentagem: Criação de fórmulas para calcular a margem de lucro, rateio de custos fixos, embalagem e preço final por unidade de rendimento.

Organização Visual: Uso de múltiplas abas interligadas e remoção de linhas de grade para criar uma interface mais limpa e focada nos dados.

🚀 Como usar
Faça o download do arquivo .xlsx.

Na aba Banco de Ingredientes, cadastre os produtos, as quantidades das embalagens e os preços pagos.

Na aba Ficha Técnica (A Receita), monte a receita listando os ingredientes. Os custos são calculados automaticamente.

Na aba Precificação, defina os custos fixos, o rendimento e a margem de lucro para visualizar o preço ideal de venda por unidade.
